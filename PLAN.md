# Metro Dashboard — Project Plan

## Goal
A publicly shareable web dashboard showing real-time Metro Trains Victoria
departure data, service disruptions, line status, and network insights.

## Stack
- Backend: Python (Flask) — proxies and signs PTV API requests, serves JSON
- Frontend: Single HTML file (HTML/CSS/Vanilla JS) — no build tools
- Hosting: GitHub Pages (frontend) + Render.com (backend)
- Repo: https://github.com/apappas57/metro-dashboard
